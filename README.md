# WEATHER STATION

WIP

A simple connected weather station made with ESP32 board. The main objective is to learn to build an IoT App with FreeRTOS.
Informations are displayed on a local screen and are also sent to a smarthpone app via Wifi or Bluetooth (TBD)

## HARDWARE

- Board: ESP32-WROOM-32D
- Temperature/Humidity sensor: DHT22
- Screen: lcd screen 2004a

## FEATURES

- Measure Temperature
- Measure Humidity
- Display informations on screen
- Send informations with WIFI to a smartphone app
